v2.1.1 -- R2C2 USB bootloader
* Turned to ON state all the stepper motors LEDs. The stepper motors are disable while on bootloader mode but the LEDs are ON. R2C2 boards shipped after 21 February 2012 uses R2C2 USB bootloader v2.1.1.

v2.1 -- R2C2 USB bootloader
* The first release of the R2C2 USB bootloader. R2C2 boards shipped after 16 February 2012 uses R2C2 USB bootloader v2.1.
